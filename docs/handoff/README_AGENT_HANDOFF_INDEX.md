## How To Use

1. Upload all files to your coding agent.
2. Start by pasting `07_AGENT_PROMPT_TO_BUILD_APP.md`.
3. Tell the agent to read every markdown file before coding.
4. Make the agent follow `05_TASKS_CHECKLIST.md`.
5. Do not move to polish until the required demo path works.

## Main Goal

Build a polished Android app that wins the hackathon by showing:

- QR attendance
- Live quiz
- AI concept explainer
- Professor insight dashboard
- Student points and leaderboard
- Clean UI
- Clean architecture
- Strong README
- Tight 2-minute demo
